Förbättrad energirådgivning och förbättrade energivanor genom Quantified Self Assisted Advisory

Quantified Self innebär att man regelbundet loggar data om sig själv, exempelvis vikt, löprundor, eller utgifter. Ett otal “appar” och datorstöd finns för detta, till exempel “Runkeeper”. Dessa har visat sig vara mycket kraftfulla för att få människor att på ett bestående sätt ändra sina vanor till det bättre. Inom detta projekt avser vi lyfta över konceptet Quantified Self till området miljö- och energivanor i syfte att understödja beteendeförändringar.

Insamlad energi- och klimatdata kan liksom för träningsappar användas för att på individ- eller gruppnivå sätta upp mål och lägga sig till med nya vanor. I detta projekt avser vi dock att gå ett steg längre och i samarbete med de kommunala energi- och klimatrådgivarna, genom Energikontoren Sverige, utveckla en “Quantified Self Assisted Advisory”, eller professionell rådgivning med hjälp av Quantified Self. Energirådgivarna kommer få ta del av insamlad data och får därmed en helhetsbild som ger förbättrade möjligheter att ge bättre råd.

A. Prototyper El

Kilowh.at

Ett verktyg för att kunna jämföra hur mycket energi som egentligen går åt för olika aktiviteter. Används för att öka kunskap i “energimedvetenhet”

Finns på: http://kilowh.at

Källkod på: Github

Energigrisen

Den här prototypen är ett samarbetsspel där personer tillsammans kan lära sig mer om hur man kan spara energi i hemmet. Det finns ett antal utmaningar sorterade på olika delar av hemmet (vardagsrum, kök etc), och ju fler utmaningar gruppen som helhet utför desto bättre går det för gruppen i spelet.

Finns på: http://energigrisen.foodprint.nu (välj “skapa nytt konto”, inloggning med google och facebook fungerar inte längre)

Energy Weight

Den här prototypen visar möjligheterna med ett “tangible interface” för att lära sig att tolka energiinformation. Den använder träklossar som läggs på en våg för att testa hur mycket solceller räcker till för eller hur mycket solceller behövs för att täcka behovet.

Källkod finns på: Github

B. Prototyper Mat

Foodprint

En plugin till webbläsare som räknar ut koldioxidavtryck av recept på nätet, t.ex. Tasteline. Det går även att mata in egna recept. Man kan även se sin utveckling över tid och jämföra med andra.

Finns på: http://foodprint.nu (välj “skapa nytt konto”, inloggning med google och facebook fungerar inte längre)

Inköpslista med klimatberäkning

En inköpslista för mobil och dator som räknar ut koldioxidavtryck av det man har på listan. Det går även att få ut vad det motsvarar i andra enheter, såsom timmar TV-tittande, kilometer bilåkning och öl.

Inköpslistan finns för tillfället på: http://www.csc.kth.se/~gustjoh/Projekt_backup/

Skolors mat-klimatavtryck

Ett system som räknar ut skolors mat-klimatavtryck baserat på data från inköpssystemet “Hantera livs” (som används av t.ex. Stockholms stad).

Prototypen kräver tillgång till rådata från ett inköpssystem och är därför inte publikt tillgängligt just nu.

Food to carbon

Ett verktyg för att kunna jämföra hur mycket koldioxid olika livsmedel ger upphov till. Används för att öka medvetenheten om olika livsmedels klimatpåverkan.

Finns på: http://food.kilowh.at
